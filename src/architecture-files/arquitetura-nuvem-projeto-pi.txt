Desenho da Arquitetura:
Frontend:

Amazon S3 (Hospedagem de sites React)
CloudFront (CDN para sites)
Backend:

EC2 ou Elastic Beanstalk (Hospedagem do SpringBoot)
API Gateway (Gerenciamento das APIs)
Lambda (Processamento sob demanda)
Integração de Pagamento:

API Gateway (Integração com Mercado Pago)
Armazenamento de Dados:

Amazon RDS (Banco de Dados MySQL)
Amazon S3 (Armazenamento de imagens e vídeos)
Monitoramento:

CloudWatch (Monitoramento e Alarme)
CloudTrail (Auditoria)
Segurança:

IAM (Gerenciamento de Permissões)
Amazon Cognito (Autenticação)
WAF (Firewall de Aplicação)
CI/CD:

CodePipeline (Entrega Contínua)
CodeBuild (Build Automatizado)
